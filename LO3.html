Leeruitkomst 3: Design

Om mijn eerste leerdoel aan te tonen heb ik verschillende dingen gedaan.

Professionele design tools

Bij het maken van mijn designs heb ik vooral gebruik gemaakt van Adobe Programma’s. Ook heb ik gebruik gemaakt van Figma voor prototypes. Figma is een handige tool om samen te werken met andere. En met groepswerk kan dit heel handig zijn. Soms gebruik ik Adobe XD maar dit doe ik alleen als ik privé werk heb en niet samen hoef te werken. Voor de opdracht Branding en voor mijn portfolio heb ik voor het logo gebruik gemaakt van Illustrator. Dit heb ik gedaan omdat deze tool speciaal bedoelt is voor logo design. Het verschil met Figma is dat je in illustrator vector bestanden kan maken. Dit werkt veel beter bij logo’s omdat je dan geen kwaliteitsverlies hebt. Voor de opdracht Branding heb ik ook stylescapes gemaakt. Deze heb ik gemaakt in InDesign. Hier heb ik voor gekozen omdat er typographie in staat. Ik heb de Designs voor mijn portfolio gemaakt in Adobe XD en InDesign. Voor de “old school” kranten designs gebruik ik Photoshop (contact knop). Als je foto’s naar jou mening wil manipuleren, is dit de perfecte tool daarvoor. Daarom heb ik deze tool hiervoor gekozen.

Iteratief ontworpen

Tijdens het maken van mijn portfolio heb ik regelmatig aan Bram gevraagd over wat zijn mening hierover is en hoe ik het zou kunnen aanpakken. Ook komen natuurlijk weer de usertesten hiervoor terug en hierdoor val ik eigenlijk de hele tijd terug op een oog buiten het project. Bij mijn portfolio heb ik eerst designs gemaakt, hier kreeg ik als feedback op dat ik het meer “mijn” stijl moest maken. Dit heb ik aangepast en weer feedback op gevraagd. De feedback die ik daar weer op kreeg was dan dat ik of vol voor een stijl moest gaan of helemaal niet. En niet ertussen in. Bij het logo design van het project Branding en ook voor het logo van mijzelf, heb ik ook regelmatig feedback gevraagd. Hierdoor kreeg ik ook veel logo’s. Bij het project Branding was het bijvoorbeeld dat ik heel vaak een heel klein beetje moest aanpassen. Toen ik dat had gedaan moest ik kijken of ik misschien in plaats van initialen, de hele naam kan pakken. Hier had ik ook weer designs van gemaakt en hier kreeg ik de feedback op dat het iets “viezer” moest en meer techno. Op deze manier ben ik op een iteratieve wijze door het design proces gegaan. Voor de stylescapes van het project Branding ben ik ook iteratief aan de slag gegaan. Ik was in het begin te lang met 1 stylescape bezig en ik probeerde het te perfect te maken. Mijn feedback was hier dat ik niet teveel moest willen meteen maar dat ik de hele tijd kleine aanpassingen maak. Dit deed ik maar het was nog steeds onduidelijk wat voor stijl het was. De feedback hier weer op was dat ik meer textuur en sfeer mee moest geven. Op het einde zijn hier hele mooie stylescapes uitgekomen met een duidelijke stijl.
